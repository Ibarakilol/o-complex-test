## Запуск проекта

1. **Клонирование репозитория:**

```bash
git clone https://github.com/Ibarakilol/o-complex-test.git
cd o-complex-test
```

2. **Установка зависимостей и запуск:**

Запуск производится без Turbopack, из-за sass пакета

```bash
npm i && npm run dev
```
